  🔧 SCRIPTS DE INSTALAÇÃO E CONFIGURAÇÃO
═══════════════════════════════════════════════════════════════════════════

  🔹 instalar_servidor.bat
     └─ Instalação automática completa
     └─ Cria ambiente virtual e instala dependências
     └─ Execute este PRIMEIRO

  🔹 configurar_firewall.bat
     └─ Libera porta 5000 no firewall
     └─ Execute como ADMINISTRADOR

  🔹 testar_configuracao.bat
     └─ Verifica se tudo está OK
     └─ Use para diagnóstico de problemas


═══════════════════════════════════════════════════════════════════════════
  🚀 SCRIPTS PARA INICIAR O SERVIDOR
═══════════════════════════════════════════════════════════════════════════

  🟢 start_server_network.bat
     └─ Atalho simples para iniciar o servidor
     └─ Clique 2x para executar
     └─ RECOMENDADO para uso diário

  🟢 server.py
     └─ Servidor Flask simples
     └─ Execute: python server.py
     └─ Bom para poucos usuários (1-10)

  🟢 server_production.py
     └─ Servidor robusto com Waitress
     └─ Execute: python server_production.py
     └─ RECOMENDADO para muitos usuários (10+)
     └─ Requer: pip install waitress

═══════════════════════════════════════════════════════════════════════════
  🎬 FLUXO DE TRABALHO RECOMENDADO
═══════════════════════════════════════════════════════════════════════════

  CONFIGURAÇÃO INICIAL (Faça UMA vez):
  
    1. 📖 Leia: COMECE_AQUI.txt
    2. 🔧 Execute: instalar_servidor.bat
    3. 🔓 Execute como Admin: configurar_firewall.bat
    4. ✅ Execute: testar_configuracao.bat
    5. 👥 Execute: criar_atalho_funcionario.bat
    6. 📤 Distribua o atalho para funcionários


  USO DIÁRIO:
  
    MANHÃ:
    1. 🚀 Clique 2x em: start_server_network.bat
    2. 📝 Anote a URL se necessário
    3. ✅ Mantenha a janela aberta

    NOITE:
    1. ⏹️ Pressione Ctrl+C no terminal
    2. 💾 Execute: fazer_backup.bat (recomendado semanalmente)


  PARA MUITOS USUÁRIOS (Performance):
  
    1. Abra terminal (cmd) na pasta
    2. Digite: venv\Scripts\activate
    3. Digite: python server_production.py
