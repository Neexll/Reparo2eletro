Olá! Você quer rodar o sistema no servidor da empresa para que seus
funcionários acessem pela rede local. Siga este guia passo a passo:


╔═══════════════════════════════════════════════════════════════════════════╗
║                   🎯 ESCOLHA SEU MÉTODO DE INSTALAÇÃO                     ║
╚═══════════════════════════════════════════════════════════════════════════╝


  ┌─────────────────────────────────────────────────────────────────────┐
  │  MÉTODO A: DISTRIBUIÇÃO EXECUTÁVEL (RECOMENDADO) ⭐                 │
  └─────────────────────────────────────────────────────────────────────┘

     VANTAGENS:
     ✓ Não precisa instalar Python no servidor
     ✓ Não precisa instalar dependências
     ✓ Basta copiar e executar
     ✓ Mais profissional e fácil
     
     PASSOS:
     
     1. NO SEU COMPUTADOR (onde está desenvolvendo):
        → Clique 2x em: build_server.bat
        → Aguarde 5-10 minutos (criando build)
        → Pasta "dist_servidor" será criada
     
     2. COPIE a pasta "dist_servidor" para o servidor
        (Use pendrive, rede, etc.)
     
     3. NO SERVIDOR:
        → Abra a pasta "dist_servidor"
        → Execute: INICIAR_SERVIDOR.bat
        → Anote a URL que aparecer
     
     4. Configure o firewall (se necessário):
        → Execute como Admin: CONFIGURAR_FIREWALL.bat


  ┌─────────────────────────────────────────────────────────────────────┐
  │  MÉTODO B: INSTALAÇÃO MANUAL (Tradicional)                         │
  └─────────────────────────────────────────────────────────────────────┘

     Use se preferir ter Python instalado no servidor.
     
     PASSO 1: Instalar
        Clique 2x em: instalar_servidor.bat
        ⏱️ Aguarde instalação concluir

     PASSO 2: Configurar Firewall
        Clique direito em: configurar_firewall.bat
        Selecione: "Executar como Administrador"

     PASSO 3: Iniciar Servidor
        Clique 2x em: start_server_network.bat
        📝 ANOTE a URL que aparecer
        ⚠️ NÃO FECHE a janela!


╔═══════════════════════════════════════════════════════════════════════════╗
║                  👥 CONFIGURAR ACESSO DOS FUNCIONÁRIOS                    ║
╚═══════════════════════════════════════════════════════════════════════════╝

  Agora que o servidor está rodando, você precisa dar acesso aos 
  funcionários. Escolha UMA das opções:


  ┌─────────────────────────────────────────────────────────────────────┐
  │  OPÇÃO A: Criar Atalho Automaticamente (MAIS FÁCIL)                │
  └─────────────────────────────────────────────────────────────────────┘

     1. Clique 2x em: criar_atalho_funcionario.bat
     
     2. Será criado o arquivo: Abrir_Reparo2Eletro.bat
     
     3. Copie este arquivo para o desktop de cada funcionário
     
     4. Funcionários clicam nele para acessar o sistema


  ┌─────────────────────────────────────────────────────────────────────┐
  │  OPÇÃO B: Enviar Link por Email/WhatsApp                           │
  └─────────────────────────────────────────────────────────────────────┘

     1. Copie a URL do servidor (ex: http://192.168.1.100:5000/dashboard)
     
     2. Envie para os funcionários
     
     3. Peça para salvarem nos favoritos do navegador

╔═══════════════════════════════════════════════════════════════════════════╗
║                         ⚠️ IMPORTANTE LEMBRAR                             ║
╚═══════════════════════════════════════════════════════════════════════════╝

  1. MANTENHA O SERVIDOR RODANDO
     ➜ Não feche a janela do terminal
     ➜ Para parar: pressione Ctrl+C

  2. MESMA REDE
     ➜ Servidor e funcionários devem estar na mesma rede
     ➜ Não funciona pela internet (apenas rede local)

  3. BACKUP
     ➜ Faça backup do arquivo "database.db" regularmente
     ➜ É onde todos os dados estão salvos


╔═══════════════════════════════════════════════════════════════════════════╗
║                      🆘 PROBLEMAS COMUNS                                  ║
╚═══════════════════════════════════════════════════════════════════════════╝

  ❌ "Funcionários não conseguem acessar"
     ➜ Execute: testar_configuracao.bat
     ➜ Verifique se estão na mesma rede
     ➜ Confirme que o firewall foi configurado

  ❌ "Porta já em uso"
     ➜ Outra aplicação está usando a porta 5000
     ➜ Feche outros programas e tente novamente

  ❌ "Erro ao instalar"
     ➜ Certifique-se que tem Python instalado
     ➜ Instale em: https://www.python.org/downloads/
